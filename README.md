## 🚀 Featured Projects

### 🗂️ [Task Manager API RESTful](https://github.com/joaomazzaropi/Task-Manager-API-RESTFul)
> RESTful task management API built with ASP.NET Core 8.

- **Clean Architecture** with Domain, Application, Infrastructure and API layers
- **CQRS** pattern with MediatR for command/query separation
- **Result Pattern** for explicit, exception-free error handling
- Input validation via **FluentValidation** integrated into the MediatR pipeline
- Full unit test coverage with **xUnit** and **Moq**
- API documented with **Swagger**

`C#` `ASP.NET Core 8` `EF Core` `MediatR` `FluentValidation` `xUnit` `SQLite`
